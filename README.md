# Plasma-Tux-Splashscreen
 
Simple Tux splash screen using a simple tux logo.

https://store.kde.org/p/2350483
 
Install through Plasma settings -> Colors & Themes -> Splash Screen -> Get New -> Search for it and install.

Or make a folder "Simple-Tux" at `~/.local/share/plasma/look-and-feel/`, move `metadata.json` & `contents/` into it.
